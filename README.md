# Dong Breaker

Lorenzo birthday Arkanoid clone. The playable static site lives at the repo root for GitHub Pages.

## Local Workflow

Preview the static site:

```bash
npm run preview
```

Then open `http://localhost:4174`.

## GitHub Pages

This repo is set up to publish the static files from the root folder. Commit the root files and push them:

```bash
git add .
git commit -m "Build Lorenzo birthday game"
git push
```

For a reminder of the Pages steps:

```bash
npm run deploy:help
```
